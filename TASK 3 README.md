# Basic Chatbot

## Project Overview

This project is a simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs and demonstrates the use of conditional statements, functions, loops, and input/output operations.

## Features

- Responds to common greetings
- Answers simple questions
- Runs continuously until the user exits
- Beginner-friendly implementation

## Technologies Used

- Python
- Functions
- If-Elif Statements
- While Loops
- User Input and Output

## How to Run

1. Install Python.
2. Open terminal in the project folder.
3. Run:

```bash
python chatbot.py
```

## Example Conversation

```
You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
```
