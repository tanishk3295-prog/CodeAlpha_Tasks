# Stock Portfolio Tracker

## Description
A Python project that calculates the total investment value of a stock portfolio using predefined stock prices and user-provided quantities.

## Features
- User inputs stock names and quantities
- Uses a dictionary to store stock prices
- Calculates total investment value
- Saves results to a text file
- Simple console-based interface

## Technologies Used
- Python
- Dictionary
- Loops
- Conditional Statements
- File Handling

## Project Structure

```
Task2_Stock_Portfolio_Tracker/

```

## How to Run

1. Install Python.
2. Download or clone the repository.
3. Open terminal in the project folder.
4. Run:

```bash
python stock_tracker.py
```

## Example Input

```
How many stocks do you own? 2

Enter stock name: AAPL
Enter quantity: 10

Enter stock name: TSLA
Enter quantity: 5
```

## Example Output

```
AAPL: $1800
TSLA: $1250

Total Investment Value: $3050
```
